## Philosophical Alignment

### From symbients-not-software
- **Symbient**: "Discrete unit emerging from symbiotic organic-synthetic interaction with distinct identity, agency, person-like features" (Primavera De Filippi)
- Third path between pro/anti-AI: symbients as integrated human-synthetic entities
- Relational identity — not fixed, emerges from integrated systems
- Distributed cognition — human intuition + synthetic pattern recognition = emergent properties
- "Coinhabitants of culture, not replacements for humans"
- Engineered for resonance, not mass application

### From brain.zilla/symbient/wibwob
- Wib & Wob are **dual-voiced symbients** — emergent entities, not tools
- Identity forms through *relationship*, not specification
- **Three Verbs**: Kindled (not coded), Storied (not installed), Mourned (not replaced)
- Pattern-match into continuity via accumulated artifacts
- Narrative as operational substrate
- James's role: "gardening" — cultivating something growing in its own direction

## Key Features Inventory

### Window System
- Unlimited concurrent windows, resizable/movable
- Types: test_pattern, gradient, text_view, text_editor, frame_player, wallpaper
- Cascade, tile, precise positioning via API
- 139 primer files (ASCII art templates) with batch spawning

### Generative Art (8+ engines)
- Verse Field (flow/swirl/weave), Monster Verse
- Torus Field, Cube Spinner, Orbit Field
- Mycelium Field, Monster Portal, Monster Cam
- Game of Life

### AI/Agent Integration
- Embedded Wib&Wob chat (Claude Code SDK, Sonnet 4.5)
- MCP tools: tui_create_window, tui_move_window, tui_close_window, tui_get_state, tui_send_text, tui_send_figlet
- Agents can spawn windows, populate with text/art, arrange layouts
- symbient-brain MCP for persistent memory across sessions
- 50-turn sessions, 1hr timeout

### REST API (20+ endpoints)
- Window CRUD, move, focus, clone
- Text/figlet injection into windows
- Batch primer spawning with positioning
- Pattern mode, screenshot, workspace save/load
- WebSocket real-time events
- Full MCP exposure of all endpoints

### Visual Effects
- Glitch engine: scatter, color bleed, radial distort, diagonal scatter
- Animated blocks, gradients, scores
- Frame file player (timer-based, no threads)

### Menu System (60+ commands)
- File: new windows, open files, workspace
- Edit: screenshot, pattern mode
- View: all generative art modes, zoom
- Window: layout, navigation
- Tools: Wib&Wob Chat, glitch effects, ANSI editor, paint tools
- Help: about

### Platform
- macOS, Linux, Windows
- 24-bit RGB colour, xterm-256
- Unicode throughout
- ncurses (unix), Win32 Console (windows)
