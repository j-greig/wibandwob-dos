# Scramble Desktop Pose Sketches

Design explorations for Scramble's TUI presence. Each shows a full desktop
context with windows, menu bar, and Scramble in a different state/position.

Cat is ~8-10 chars wide, sits ON the desktop as a transparent overlay (not a window).

---

## Pose Matrix (summary)

| State | Kaomoji | Size | Placement | Frequency |
|-------|---------|------|-----------|-----------|
| sleeping on frame | `/ᐠ- -ᐟ\-zzz` | medium | draped over window top | idle >10 min |
| corner idle | `/ᐠ｡ꞈ｡ᐟ\` | small | bottom-right corner | default |
| curious | `/ᐠ°ᆽ°ᐟ\` | medium | near active art window | art engine opens |
| menu perch | `/ᐠ｡ꞈ｡ᐟ\` | micro | on menu bar | random rare |
| deep loaf | `/ᐠ- -ᐟ\-zzz` | wide/flat | between windows | long idle |
| belly up | `\ᐟ｡ꞈ｡ᐠ/` | medium | centre of empty desktop | post-feed |

## Rendering Notes

- Cat is a transparent overlay on the desktop — windows render behind/under her
- Speech bubble clips to screen bounds, repositions if near edge
- Pose transitions are instant (cats don't animate, they teleport)
- Z-order: Scramble is always topmost except for active modal dialogs
- When hidden: cat fades to just ears poking above status line, then gone
